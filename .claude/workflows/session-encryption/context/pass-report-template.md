# Pass Report

When implementation and tests pass, write a pass report to
`.claude/workflows/session-encryption/working/pass-reports/<N>-<slug>.md`.

The report includes a **Pass-forward** section: read the next pass's `prompt.md`
first, then write only what that agent needs from this pass (3–8 bullets).

**Before writing the report**, add a todo item:
"Read `.claude/workflows/session-encryption/source/pass-report-template-full.md` for the full template structure."
Do NOT skip this — the report must follow the exact template.
