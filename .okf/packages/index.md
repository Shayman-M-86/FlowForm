# Shared packages

Workspace packages consumed as source (no build/publish step) by both
[studio-app](/apps/studio-app.md) and [public-site](/apps/public-site.md).

* [@flowform/ui](ui.md) - 18 shared, app-agnostic UI components
* [@flowform/styles](styles.md) - design tokens, Tailwind v4 config, fonts
* [@flowform/site-shell](site-shell.md) - shared SiteHeader and nav constants
* [@flowform/builder](builder.md) - survey editor and form-filler runtime
