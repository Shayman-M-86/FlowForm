# FlowForm Packages — Claude Code Guide

## What this is

Shared workspace packages consumed by `studio-app` and `public-site`.

## Packages

| Package | Purpose |
|---|---|
| `@flowform/ui` | Shared UI components (Button, Card, Modal, Badge, Spinner, etc.) |
| `@flowform/styles` | Design tokens, Tailwind v4 config, component utility classes |
| `@flowform/site-shell` | SiteHeader CSS and nav link constants |
| `@flowform/builder` | Survey/question builder widget |
